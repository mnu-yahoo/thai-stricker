# Stitch Design Notes

- Design will be provided later from Stitch
- Do not invent final UI
- App should look professional, modern, and focused
- Avoid implementing screens before design direction is provided

## Theme Feature

Thai Stricker supports two app themes:

- Light
- Dark

Themes are intended to be generated from Google Stitch.

Until Stitch final tokens are available, the app may use placeholder theme tokens.

Theme tokens live in:

`src/styles/theme.ts`

The Settings screen Theme preference controls which theme is active.
