# Google Stitch Prompt: Thai Stricker Light and Dark Themes

Design two professional mobile app color themes for a private Muay Thai training app called Thai Stricker.

The app is Android-first and built with React Native Expo.

Create:
1. Light Theme
2. Dark Theme

Brand feel:
- Modern
- Athletic
- Focused
- Premium but not flashy
- Inspired by Muay Thai training, discipline, and fight-camp energy
- Suitable for daily personal training

Avoid:
- Medical or fitness-clinic look
- Overly aggressive fight poster style
- Neon overload
- Low contrast text
- Colors that hurt readability

Use cases:
- Home dashboard
- Workout cards
- Calendar preview
- Settings screen
- AI Coach card
- Start workout timer screen
- Add/Edit workout forms

For each theme, provide design tokens for:

- appBackground
- surface
- surfaceMuted
- card
- cardElevated
- textPrimary
- textSecondary
- textMuted
- border
- primary
- primaryPressed
- primaryText
- accent
- success
- warning
- danger
- navbarBackground
- navbarActive
- navbarInactive
- inputBackground
- inputBorder

Also provide:
- recommended button style
- card style
- input style
- calendar day style
- active tab style
- typography guidance

Output should be practical for React Native StyleSheet integration.

Return color values as hex codes.
